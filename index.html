Digital Music Box

A mechatronic song composer

Software and hardware:

  • Python + Kivy
  • Arduino Mega & Raspbery Pi + SlushEngine
  • 200+ fabricated, COTS, and stock parts
  • ODrive motor, capactivate sensors, RGB LEDs
  • MIDI protocol

Features:

  • Allows for different notes, instruments, tempos
  • Capacitavely senses movable metal pins
  • Mixes tracks with a MIDI sound module
  • Manages data with Arduino + Raspberry Pi
  • WIP: Touch-screen interface
Video Source

Digital Music Box prototype. The prototype has 3 tracks available for composition with metal pegs on the drum, which trigger the MIDI module.

Digital Music Box full model. The final product will have a removable drum with 12 programmable tracks and a touch-screen GUI.

Pi Habits

An IoT habit tracker

Software and hardware:

  • Python + Flask
  • JS + jQuery
  • Raspbery Pi + Unicorn HAT
  • CSS3 + Bootstrap
  • HTML5

Features:

  • Online heatmap syncs with LED grid
  • SQL database tracks users, habits, history
  • Activity frequency reflects in color, brightness
  • Raspberry Pi hosts site server
  • Tracks and indicates streaks
Video Source

The PiHabits IoT habit tracker is fully responsive and gives users a platform to log their habit activity from any device.

PiHabits displays a unique activity heatmap for each user on its site and on an LED grid. The intensity and color of the LEDs indicates activity frequency.
